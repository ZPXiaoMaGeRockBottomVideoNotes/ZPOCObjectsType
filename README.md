# ZPOCObjectsType
本Demo主要介绍了OC对象的三个种类：instance对象（实例对象）、class对象（类对象）、meta-class对象（元类对象）以及他们中存储的内容，还介绍了object_getClass和class_isMetaClass函数的用法。
